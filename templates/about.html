About This Project

This Iris Classifier is a simple machine learning web application developed using Python, Flask, and the scikit-learn library.

It uses a Random Forest Classifier to predict the species of an Iris flower based on its petal and sepal dimensions.

This project is created as part of an internship submission to demonstrate practical knowledge in:

  • Machine Learning
  • Flask Web Development
  • Model Deployment
  • HTML Templating with Jinja2
